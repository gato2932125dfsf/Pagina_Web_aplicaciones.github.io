¿Qué es GitHub?

GitHub es una plataforma en línea que combina el control de versiones de Git con herramientas colaborativas. Permite a los desarrolladores trabajar juntos, guardar proyectos en la nube y mantener un historial completo de cambios.

GitHub es más que una herramienta técnica: es una comunidad donde millones de personas comparten conocimientos, aprenden y colaboran en proyectos innovadores.

Manual de GitHub

Este manual te guiará paso a paso para crear tu cuenta, configurar tu primer repositorio y comenzar a subir tus proyectos.

  • 1️⃣ Crear una cuenta: Entra a github.com y haz clic en Sign Up.
  • 2️⃣ Crear un repositorio: Define un nombre, descripción y si será público o privado.
  • 3️⃣ Clonar el repositorio: Usa git clone [URL] para copiarlo a tu computadora.
  • 4️⃣ Guardar cambios: Ejecuta git add . y git commit -m "mensaje".
  • 5️⃣ Subir a GitHub: Con git push origin main sincronizas tu trabajo con la nube.
